Commit

Um commit registra uma alteração local no repositório. Ele cria um "ponto de salvamento" do estado atual dos arquivos.

Exemplo de uso: Depois de modificar arquivos, você faz um commit com uma mensagem que explica o que foi alterado

Importante: O commit ainda não envia as alterações para o repositório remoto (como o GitHub); ele só salva localmente.
